Solid State Drives (SSD)

Advantages:

• Faster

• Lighter

• More reliable (no moving parts)

• Smaller

• Low power consumption

Disadvantages:

• High cost per GB

• Limited amount of write cycles

How they work:

There are 2 types of SSD gates, NAND and NOR. NAND gate SSDs are cheaper to make but are a bit slower than NOR gate SSDs

NAND gate SSDs use flash memory cells which work by "trapping" an electron in a floating gate, hence the name "flash memory".

NOR gate SSDs are called EEPROMS.

NOTE: nowadays both NOR and NAND SSDs use flash memory cells, but the book says only NAND uses them.
